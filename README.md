gomorse
========

> **gomorse** is a simple little command line utility to convert stdin input into morse code.

Run
------------------------------

To build the executable run:, 

    $ go build macmorse.go

To convert string to morse, try:

    $ macmorse test

where `test` is a sample.

To run tests:

    $ go test
